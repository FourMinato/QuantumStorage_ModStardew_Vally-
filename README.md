# 💠 QuantumStorage - Stardew Valley Mod

มอดพอร์ทัลเก็บของข้ามมิติที่เชื่อมโยงหีบทั้งหมดในฟาร์มของคุณเข้าด้วยกัน ผ่านอินเตอร์เฟซที่สวยงามและชาญฉลาดที่สุด (Smart Interface).

## ✨ คุณสมบัติเด่น (Features)
- **Universal Access:** เข้าถึงและจัดการไอเทมในหีบทั้งหมดบนฟาร์มได้จากเมนูเดียว ไม่ว่าจะอยู่ในเหมืองหรือชั้นไหนก็ตาม
- **Smart Integration:** รองรับการดึงไอคอนหมวดหมู่โดยตรงจากมอด **SmartChest** เพื่อแสดงผลบนหน้าต่าง UI โดยอัตโนมัติ
- **Dynamic Positioning:** ระบบ UI ออกแบบมาให้เกาะติดกับตัวหีบแบบ Responsive 100% ไม่หลุดขอบจอแม้ย่อหน้าต่างหรือเปลี่ยน Zoom Level
- **Location Sync:** แสดงชื่อสถานที่ (Location) ของหีบแต่ละใบเพื่อให้คุณจัดการคลังแสงได้อย่างแม่นยำ

## 🛠️ ความต้องการ (Requirements)
- [SMAPI - Stardew Modding API](https://smapi.io/)
- Stardew Valley เวอร์ชั่น 1.6 ขึ้นไป
- [GenericModConfigMenu](https://www.nexusmods.com/stardewvalley/mods/5098)
- *[แนะนำ]* **SmartChest** (หากต้องการฟังก์ชันการแสดงไอคอนหมวดหมู่)

## 🚀 วิธีการติดตั้ง (Installation)
1. ติดตั้ง SMAPI เวอร์ชั่นล่าสุด
2. แตกไฟล์มอดลงในโฟลเดอร์ `Mods`
3. เข้าเกมผ่าน SMAPI และเพลิดเพลินกับการเปิดมิติใหม่แห่งการเก็บของ!
